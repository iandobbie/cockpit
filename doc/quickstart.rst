Cockpit Quick Start Guide
*************************

Once installed can be run typing *cockpit*. With no configuration
cockpit will start up with a default set of simulated hardware
attached so that you can test the system with no real hardware
attached.

Alternatively, starting cockpit with the *--no-config-files* flag will
acheive the same effect::
  cockpit --no-config-files
